# ES添削補助ツール

エントリーシート（ES）を貼り付けるだけで、文字数・文章構造・弱い表現を自動チェックするWebツールです。

**▶ デモ: https://＜あとでGitHub PagesのURLに差し替え＞**

## 機能

- **文字数カウント** — 空白・改行を除いた実質文字数／目標文字数に対する達成率をリアルタイム表示
- **構造チェック** — 段落数・文の数・一文の平均文字数を計測し、60字を超える長文を検出
- **弱い表現の検出** — 「思います」の連発、「頑張った」「様々な」などの抽象語・ぼかし語を指摘
- **結論ファースト判定** — 冒頭で結論・テーマが示されているかを簡易チェック

## 特徴

- **完全ローカル処理** — 入力したESはブラウザ内でのみ処理され、外部に一切送信されません
- **インストール不要** — ブラウザで開くだけ。依存ライブラリゼロの単一HTMLファイル

## 使い方

1. ページを開く
2. ESの本文を貼り付ける
3. 目標文字数を設定する（例: 400字）
4. 右側に表示される診断結果をもとに推敲する

## 背景

就活で数十枚のESを書く中で「提出前の機械的なセルフチェック」を毎回手作業でやっていたため、自動化するために開発しました。

## 技術

HTML / CSS / JavaScript（Vanilla）— フレームワーク・ビルド不要の構成
